Project Summary

This project was to make an easy & efficient way to enhance blurry images using iOS. This includes:

  • A react-native front-end which allows image & model selection
  • An AWS S3 Bucket for transferring images between the front-end & the replicate API
  • Three AWS Lambda Functions to process & return the image to the front-end
  • Secure integration between AWS Services and the React Native App

Project Reflections & Improvements

This project allowed me to learn tech stacks which I was unfamiliar with e.g. Python, Java, AWS Services, APIs, HTML & CSS.
While, there is more to do, the app is at a functional stage with all the other components integrated.
